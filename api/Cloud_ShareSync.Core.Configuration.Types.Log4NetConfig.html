Class Log4NetConfig

Cloud-ShareSync instruments logging via a custom ILogger implementation (ref TelemetryLogger) that utilizes apache Log4Net as its backend.

The built in/default Log4Net configuration consists of a rolling log file appender and colored console appender for all standard log messages. OpenTelemetry traces are also exported, in json format, to a rolling log file appender.

These built in settings can also be optionally overridden via a log4net XML configuration file.
