FileName

The file name and extension for the telemetry rolling log file.

If (RolloverCount > 0) then logs will roll over and be renamed with the format "FileName.#.Extension" once the MaximumSize has been met.

Default value set to "Cloud-ShareSync-Telemetry.log".
