CompressFile(FileInfo, PrimaryTable)

Checks whether compression is enabled.
If compression is not enabled then returns the inputFile.
If compression is enabled then

  1. Optionally create a Unique Compression Passwords.
  2. Compress inputFile using 7z CompressionInterface.
  3. Add compression data to database.
  4. Delete inputFile.
  5. return compressed FileInfo object.

EncryptFile(FileInfo, PrimaryTable)

Checks whether encryption is enabled.
If encryption is not enabled then returns the inputFile.
If encryption is enabled then

  1. Generate cypherTxtFile FileInfo object.
  2. Generate random 32 byte crypto key.
  3. Encrypt inputFile data into cypherTxtFile.
  4. Add DecryptionData to database.
  5. Delete inputFile.
  6. return cypherTxtFile.
